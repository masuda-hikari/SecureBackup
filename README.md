# SecureBackup

差分・暗号化バックアップツール - 大切なデータを安全に自動バックアップ

## 特徴

- **差分バックアップ**: BLAKE3ハッシュによる高速差分検出、変更ファイルのみをバックアップ
- **AES-256-GCM暗号化**: 軍事グレードの暗号化でデータを保護
- **Zstd圧縮**: 高効率圧縮でストレージを節約
- **直感的なUI**: シンプルで使いやすいデスクトップアプリ
- **ダークモード**: 目に優しいダークテーマ対応

## インストール

### Windows

1. [Releases](https://github.com/mmasu/SecureBackup/releases) から最新版をダウンロード
2. インストーラーを選択:
   - `SecureBackup_x.x.x_x64-setup.exe` - NSISインストーラー（推奨）
   - `SecureBackup_x.x.x_x64_en-US.msi` - MSIインストーラー

### 動作要件

- Windows 10/11 (64-bit)
- WebView2 Runtime（自動インストール）

## 使い方

### バックアップ

1. アプリを起動
2. 「バックアップ元」でバックアップしたいフォルダを選択
3. 「バックアップ先」で保存先フォルダを選択
4. （任意）暗号化オプションを有効化し、パスワードを設定
5. 「バックアップ開始」をクリック

### 復元

1. 「復元」タブに切り替え
2. 「バックアップフォルダ」でバックアップ済みフォルダを選択
3. 「復元先」で復元先フォルダを選択
4. （暗号化バックアップの場合）パスワードを入力
5. 復元したいファイルを選択
6. 「復元開始」をクリック
