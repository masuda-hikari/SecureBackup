# Changelog

All notable changes to SecureBackup will be documented in this file.

## [0.1.0] - 2026-01-18

### Added
- **バックアップ機能**
  - フォルダ選択ダイアログによるソース・宛先指定
  - BLAKE3ハッシュによる差分検出（変更ファイルのみバックアップ）
  - AES-256-GCM暗号化対応（パスワード保護）
  - Zstd圧縮によるストレージ効率化
  - リアルタイム進捗表示
  - パスワード強度チェック

- **復元機能**
  - バックアップ情報の読み込み・表示
  - ファイル一覧表示と個別選択
  - 全選択/全解除機能
  - 暗号化ファイルの復号
  - 圧縮ファイルの解凍
  - 上書きオプション
  - 復元進捗表示

- **UI/UX**
  - タブ切り替え（バックアップ/復元）
  - ダークモード対応
  - レスポンシブデザイン
  - エラーメッセージ表示

### Technical
- Tauri 2.0 (Rust + React/TypeScript)
- Windows x64対応
- 10件のユニットテスト全PASS
