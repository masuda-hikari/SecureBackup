## 必須チェック項目（各セッション終了時）

### 1. Nielsen 10 Heuristics
- [ ] システム状態の可視性（ローディング/成功/失敗表示）
- [ ] 一貫性と標準（UI要素の統一）
- [ ] エラー防止と回復支援
- [ ] 美的でミニマルなデザイン

### 2. アクセシビリティ
- [ ] コントラスト比 4.5:1以上
- [ ] タップターゲット 44dp以上（モバイル）
- [ ] キーボード操作対応（デスクトップ）

### 3. パフォーマンス
- [ ] 応答時間 0.4秒以内（超過時はローディング表示）
- [ ] 初期表示 3秒以内

### 4. プラットフォーム準拠
- [ ] iOS: Apple Human Interface Guidelines
- [ ] Android: Material Design 3
- [ ] Windows: Windows App SDK / Fluent Design
- [ ] macOS: macOS Human Interface Guidelines

### 5. 日本市場対応
- [ ] ダークモード対応
- [ ] 可読性（テキストサイズオプション）
- [ ] 日本語フォント最適化
